### A New Story 🎲

あの銀河系外も有名な戦い、聞いたことがある？\
小雨天とイワンという女の子との熾烈な運命の戦いの末は一体どのようになった、知ってる？\
これは、彼女がまだあの伝説中の百万ドルの夜景に一攫千金といった映画のような世界、アメリカのラスベガスにいる時代の物語である。
